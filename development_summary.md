# 長安 (Choan) ウェブサイト更新 開発ログ

**最終更新日:** 2026年3月27日

## 1. プロジェクト概要
既存の「長安」ウェブサイト (`index.html`) をベースに、デザインの刷新、ニュース機能の動的化、および「仕出し弁当」「ケータリング」の詳細ページ作成を行いました。また、検索エンジンにインデックスされないよう対策を施しました。

## 2. 実装・変更内容

### 共通設定
- **カラーテーマの変更**:
  - ベースカラーを `#FCEAB5` (薄いクリーム色) に変更し、「Japanese Modern / Warm Dining」の雰囲気を強化。
- **SEO/クローラー対策**:
  - 全ページの `<head>` タグ内に `<meta name="robots" content="noindex, nofollow">` を追加。
- **ロゴ設定**:
  - テキストロゴから画像ロゴ (`choan-20240207.png`) に変更。
  - ヘッダー内のロゴサイズを `80px` に拡大調整。

### ファイル別詳細

#### 1. トップページ (`index.html`)
- **新着情報 (News) セクション**:
  - 静的なHTML記述から、Google Sheets (CSV形式) を読み込んで動的に表示するJavaScript実装に変更。
  - **データソース**: `https://docs.google.com/spreadsheets/d/1nGbJXqi7QZn3OZgdBYRoKTXLa-MqDXrXmmo6ENQeaAo/export?format=csv&gid=1839177219`
  - **表示制御**: スプレッドシートの A列 (`表示chk`) を参照し、`TRUE` の記事のみを表示するようにフィルタリングを実装。
  - **インスタグラム連携**: スプレッドシートの L列 に URL がある場合、記事カードに「インスタグラムで見る」リンクを動的に表示。
  - カテゴリごとのフィルタリング、表示件数制御、フェードインアニメーションを実装。
- **サービスセクション**:
  - 「オードブル・仕出し弁当」のリンク先を `index-order.html` に変更。
  - 「ケータリング」のリンク先を `index-catering.html` に変更。
- **フビコン**:
  - `choan-20240207.png` を設定。
- **スパム対策**:
  - `index.html` のメールアドレス表示 (`info@choan2023.com`) を JavaScript による動的生成に変更し、ボットによる自動収集を防止。

#### 2. 仕出し弁当・オードブル詳細ページ (`index-order.html`)
- **新規作成**: PDF配布の予定を変更し、HTMLページとして実装。
- **コンテンツ**:
  - 「Ristorante Aico × co」コラボレーション情報の掲載（ロゴ、キャプション付き）。
  - 各メニュー（松花堂弁当、梅弁当A、華弁当、大皿オードブルL、円形オードブルM）の画像と詳細仕様（価格、内容、容器サイズ）を掲載。
  - 詳細なオプションメニューと価格表。
- **機能**:
  - ページ下部に「トップページへ戻る」ボタンを設置。
  - レスポンシブ対応のレイアウト（スマホでは1列、PCではグリッド表示）。

#### 3. ケータリング詳細ページ (`index-catering.html`)
- **新規作成**: 新たなサービスの柱としてページを構築。
- **コンテンツ**:
  - プロのお料理を届けるコンセプトメッセージ。
  - 「Aプラン」「Bプラン」「Cプラン」の詳細メニュー（価格、品数、料理内容）。
  - ドリンクプラン（ソフトドリンク、スタンダード）の詳細。
  - オプションメニュー一覧。
  - 「Ristorante Aico × co」コラボレーション情報の掲載。
- **機能**:
  - ページ下部に「トップページへ戻る」ボタンを設置。

#### 4. お問い合わせページ・通知設定 (`inquiry.html`, `form_handler.gs`)
- **inquiry.html**: 
  - デザインをサイトテーマに合わせて統一。
  - Google Apps Script (GAS) 経由での送信機能を実装。
- **form_handler.gs**: 
  - メールの送り主（表示名）を「長安問合せフォーム」にカスタマイズ。
  - スプレッドシートへの書き込み時、電話番号の先頭に `'` を付加し、先頭の「0」が消えないように修正。

#### 5. マッチング詳細ページ (`index-matching.html`)
- **UI調整**:
  - ヒーロー画像の背景表示位置を `center 29%` に調整し、人物の顔が見切れないように修正。
  - 「長安が選ばれる理由」セクションのカード内テキストを中央揃えから左寄せに変更し、可読性を向上。
  - ページ下部のリンク先を `index-matching.html` に修正。

## 4. 補足事項・今後の確認点
- **ブラウザ環境**: 現在の作業環境ではブラウザ（Chrome等）を使用したPDF解析などが利用できないため、コンテンツはご提供いただいたテキスト情報を元に構成しています。
- **画像パス**: すべての画像は `./assets/` フォルダ内にあることを前提としています。表示されない場合はファイル名や拡張子（`.png` / `.jpg`）をご確認ください。
